My Projects

QR Authentication System - Drinks

Completed • Deployed

A fun project I decided to create for my classmates' entrepreneurship week stall. Built a web application featuring a secure authentication system with a login page requiring email and password. After logging in, users can view their drinks product catalog including product logo, name, tagline, and more. The platform includes navigation links for research references, a Firebase realtime feedback system, team information, and a logout page. I placed a QR code (containing embedded login credentials) above their booth so customers could scan and instantly access the platform. Successfully deployed on Netlify.

HTML5 CSS3 JavaScript QR Code Firebase Netlify
